## How to Run the Program

To compile and run the program, follow these steps:

### Compilation

Use the following command to compile the program:

```bash
gcc -O3 main.c -o studentID
```

### Running the Program
To encrypt a file:

```bash
./studentID "e" <key file> <plaintext file> <ciphertext file>
```
